# Селектори CSS (CSS Selectors Specificity)

Цей проєкт є простою веб-сторінкою, яка демонструє різні типи CSS-селекторів та їхню специфічність (вагу). Він стане у нагоді як зручна візуальна шпаргалка для початківців у веб-розробці, щоб краще розуміти, як саме браузер обчислює пріоритетність застосування CSS-стилів.

## Особливості

На сторінці наочно продемонстровано 10 прикладів селекторів:

1. **Селектор тегу** (вага: 0-0-1)
2. **Селектор класу** (вага: 0-1-0)
3. **Селектор ID** (вага: 1-0-0)
4. **Селектор атрибуту** (вага: 0-1-0)
5. **Вкладений селектор** (вага: 0-0-2)
6. **Дочірній селектор** (вага: 0-0-2)
7. **Послідовність селекторів** (вага: 0-1-1)
8. **Комбінація тег + клас + атрибут** (вага: 0-2-1)
9. **Комбінація ID + клас** (вага: 1-1-0)
10. **Суміш всіх селекторів** (вага: 1-2-2)

Кожен приклад має власне колірне оформлення для зручності візуального сприйняття.

## Технології

- **HTML5** для семантичної розмітки сторінки
- **CSS3** для стилізації, використання сучасних властивостей (`border-radius`, `background-clip` та ін.)
